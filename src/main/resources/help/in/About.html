Program tambahan yang digunakan:

JabRef uses the Material Design Icons, which are licensed under the SIL Open Font License, Version 1.1.

JabRef menggunakan JGoodies Looks dan JGoodies Forms, yang disebarkan oleh JGoodies (http://www.jgoodies.com) mengikuti syarat dari Lisensi (Lihat http://www.opensource.org/licenses/bsd-license.html untuk perincian lanjut).

JabRef menggunakan ANTLR parser generator untuk mengenali string pencarian. ANTLR adalah perangkat lunak umum - lihat http://www.antlr.org/license.html untuk syarat lisensi.

Untuk tampilan tabel dan penapis, JabRef menggunakan Glazed Lists library (http://publicobject.com/glazedlists), yang disebarkan mengikut Lisensi Lesser General Public.

JabRef juga menggunakan threading library Spin, yang disebarkan mengikut syarat Lisensi Lesser General Public .

Beberapa komponen GUI digunakan dari Microba library, mengikut lisensi BSD style. Lihat http://sourceforge.net/projects/microba/ untuk informasi lanjut.

Hampir semua ikon berasal dari Crystal Clear dibuat oleh Everaldo (http://www.everaldo.com), yang disebarkan mengikut syarat Lisensi Lesser General Public.

Perintah baris tambahan dijalankan menggunakan Apache Commons CLI (http://commons.apache.org/cli/). License: Apache License.

Apache Commons Logging (https://commons.apache.org/proper/commons-logging/) is used for Logging. License: Apache License.

Jersey is used for the connection to Mr. DLib. License: GPLv2 with classpath exception (http://glassfish.java.net/public/CDDL+GPL_1_1.html)
