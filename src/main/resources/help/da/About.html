Tredjepartssoftware anvendt:

JabRef bruger JGoodies Looks og JGoodies Forms, distribueret af JGoodies (http://www.jgoodies.com) under betingelserne i BSD License (se http://www.opensource.org/licenses/bsd-license.html for detaljer).

JabRef bruger ANTLR parser-generatoren til fortolkning af søgestrenge. ANTLR er public domain software - se http://www.antlr.org/license.html for licensbetingelser.

Til tabelpræsentation og -filtrering benytter JabRef Glazed Lists biblioteket (http://publicobject.com/glazedlists), som distribueres under Lesser General Public License.

JabRef benytter også trådbiblioteket Spin, som distribueres under betingelserne i Lesser General Public License.

Der er brugt visse GUI-komponenter fra Microba-biblioteket, som er licenseret under en BSD-lignende licens. See http://sourceforge.net/projects/microba/ for mere information.

Størstedelen af ikonerne stammer fra Crystal Clear ikonsættet af Everaldo (http://www.everaldo.com), som distribueres under betingelserne i Lesser General Public License.

Kommandolinjeparametre fortolkes med Apache Commons CLI (http://commons.apache.org/cli/). License: Apache License.

Apache Commons Logging (https://commons.apache.org/proper/commons-logging/) is used for Logging. License: Apache License.

Jersey is used for the connection to Mr. DLib. License: GPLv2 with classpath exception (http://glassfish.java.net/public/CDDL+GPL_1_1.html)

JabRef uses the FontBox, JempBox, and PDFBox libraries for PDF handling, which are distributed under the Apache License. See http://pdfbox.apache.org for more information.
